## Other snippets

## CSS Themes

CSS snippets live in `.obsidian/snippets/`. All can be toggled in **Settings → Appearance → CSS Snippets**.

### Notebook Backgrounds (`Notebook Backgrounds.css`)
Apply per-note via `cssclasses` in frontmatter. Mix page and pen classes freely.

**Page backgrounds:**
  
| Class | Appearance |
|---|---|
| `page-white` | Clean white page |
| `page-manila` | Warm tan / manila paper |
| `page-blueprint` | Blueprint blue with white ink |

**Pen colors** (override default ink color):
`pen-black` `pen-gray` `pen-red` `pen-green` `pen-blue` `pen-purple` `pen-white`

**Modifiers:**
`page-grid` — adds grid lines to the page background  
`recolor-images` — recolors transparent-background images to match pen color  
`embed-manila` / `embed-white` / `embed-blueprint` — sets background behind embedded images

### Daily Note Themes (`Daily Note Themes.css`)

  

Seven color schemes, one per day of the week. Applied automatically by the Daily Enhanced template via Templater.

  

| Day | Primary Color |
|---|---|
| Sunday | Rose / Pink |
| Monday | Red / Coral |
| Tuesday | Blue |
| Wednesday | Yellow |
| Thursday | Orange |
| Friday | Green |
| Saturday | Crimson |
  
Requires `cssclasses: [daily, <weekday>]` in frontmatter.

### Colored Sidebar (`Colored Sidebar Items.css`)

Colors top-level folders by numeric prefix. The color cascade applies to all child files.

| Prefix | Color |
|---|---|
| `00` | Mint |
| `01` | Cyan |
| `02` | Light Blue |
| `03` | Blue |
| `04` | Violet |
| `05` | Purple |
| `06` | Magenta |
| `07` | Hot Red |

| `99` | Cool Gray |

### General Tweaks (`CyanVoxel's General Tweaks.css`)

Utility classes available on any note:
`image-borders` — adds border around images  
`center-images` — centres all images  
`center-titles` — centres all headings  
`no-embed-border` — removes embed border  

### Theme
Base theme: **Vanilla AMOLED** (pure black background in dark mode).  
Interface and editor font: **JetBrains Mono Nerd Font Mono**.

## External Integrations
  
| Tool | Role |
|---|---|
| **Zotero** | PDF archival, annotation, and research paper management. Export to Obsidian via Better BibTeX + Zotero Integration plugin. |
| **Research Rabbit** | Paper discovery and citation mapping |
| **Connected Papers** | Visual paper relationship graphs |
| **Raindrop** | Article save-for-later and reading queue |
| **Snipd / Airr** | Podcast timestamped annotation |
| **Memex** | YouTube annotation with timestamp links |
| **GitHub** | Vault sync between devices via Obsidian Git plugin |


**Zotero highlight colour convention:**
  
| Colour | Meaning |
|---|---|
| Yellow | Important point by author |
| Green | Important to me |
| Pink | Disagree with author |
| Purple | Literary note to look up later |
| Blue | Notes added later |
